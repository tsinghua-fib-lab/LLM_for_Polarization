# SI minimal data and codes

This folder contains the SI result tables and plotting code.

- `data/`: aggregated CSV result files.
- `SI_codes.ipynb`: plotting notebook that reads only files in `data/`.
- `figures/`: PDF outputs generated by `SI_codes.ipynb`.

File groups:

- `si_fig1_*`: SI panels from `fig1/fig1.ipynb`.
- `si_fig2_*`: pairwise opinion, error-frequency, and retry-frequency SI panels from `fig2/fig2.ipynb`.
- `si_fig4_*`: control-mechanism SI panels from `fig4/fig4.ipynb`.
- `si_intervention_*`: intervention SI panels from `fig5/Echo_vis/intervention_experiments.ipynb`.
- `si_nc_*`: NC experiment SI panels from `NC_experiments/experiments.ipynb`.
- `si_argument_*`: argument-extraction SI panels from `NC_experiments/argument_extraction.ipynb`.
- `si_affection_*`: affection/perception SI panels from `NC_experiments/affection.ipynb`.
